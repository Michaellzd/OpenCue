# OpenCue

A macOS teleprompter app that lives in the MacBook notch. Invisible to screen capture — your audience on Zoom, Meet, or Teams never sees it.

## Local Use

If you want to open-source the project and use it yourself, you do not need notarization first.

Build a local app bundle:

```bash
./scripts/build-local-release.sh
```

Package a simple local DMG:

```bash
./scripts/make-local-dmg.sh
```

Full instructions are in [doc/local-build-and-use.md](/Users/michaellzd/Documents/indie_project/opencue/doc/local-build-and-use.md).
